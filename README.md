# ✈️ Flight Price Analysis (EDA)

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on a flight price dataset to understand factors affecting ticket prices.

## 📊 Key Steps

* Data Cleaning and Handling Missing Values
* Feature Engineering (Date, Time, Duration)
* Data Visualization using Matplotlib & Seaborn
* Correlation Analysis

## 🔍 Insights

* Airlines have different pricing patterns
* Flights with more stops tend to cost more
* Duration has a significant impact on price

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
